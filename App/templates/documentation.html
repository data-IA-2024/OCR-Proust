Documentation de l'application et guide utilisateur Version 1.0

Bienvenue dans l'application PROCR !

Cet outil a été conçu pour vous aider à traiter vos factures et analyser vos chiffres rapidement.

1. Fonctionnalités Principales

1.1. Importer un Fichier

Cette fonctionnalité vous permet de charger des fichiers contenant le texte que vous souhaitez extraire.

    Étape 1 : Cliquez sur l'onglet "Importer un fichier" dans le menu de navigation principal.

    Étape 2 : Une page d'importation s'affichera. Cliquez sur le bouton "Importer votre fichier".

    Étape 3 : Une fenêtre de dialogue s'ouvrira. Sélectionnez le fichier que vous souhaitez importer depuis votre ordinateur.

    Étape 4 : Une fois le fichier sélectionné, cliquez sur le bouton "Lancer OCR" pour lancer le processus d'importation et d'OCR.

    Étape 5 : Une fois le traitement terminé, vous pourrez généralement consulter les résultats dans la section "Base de données".

1.2. Base de Données

La section "Base de données" vous permet de visualiser et de gérer les données extraites par l'application.

    Navigation : Cliquez sur l'onglet "Base de données" dans le menu principal.

    Affichage des données : Vous devriez voir un tableau affichant les informations extraites.

    Recherche : Un champ de recherche est disponible pour vous aider à trouver des informations spécifiques. Entrez votre requête dans l'onglet "search" et validez.

1.3. Monitoring

La section "Monitoring" vous offre une vue sur l'état et le fonctionnement de l'application.

    Accès : Cliquez sur l'onglet "Monitoring" dans le menu principal.

    Informations affichées : Sur cette page, vous pourrez consulter des informations sur la réussite de l'OCR.

1.4. Statistiques

Le tableau de bord des statistiques vous présente des informations clés sur l'utilisation de l'application et les données traitées.

    Accès : Cliquez sur l'onglet "Statistiques" dans le menu principal.

    Statistiques disponibles : Vous y trouverez des données telles que :

    • Nombre d'utilisateurs uniques.
    • Nombre total de factures traitées.
    • Chiffre d'affaires total (si applicable).
    • Panier moyen (si applicable).
    • Clients les plus prolifiques.
    • Clients les plus réguliers.
    • Villes les plus demandeuses.
    • Articles les plus achetés (si applicable).
    • Nombre de factures avec un total nul.

1.5. Documentation

La section "Documentation" fournit des informations plus détaillées sur l'application et son utilisation et vous êtes en train de la lire.

2. Déconnexion

Lorsque vous avez terminé d'utiliser l'application, il est important de vous déconnecter pour des raisons de sécurité. Une déconnexion automatique aura lieu après 30 minutes d'inactivité.

    Étape 1 : Cliquez sur le bouton "Déconnexion".

    Étape 2 : Vous serez alors redirigé vers la page de connexion.
